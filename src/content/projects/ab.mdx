# AI Task Manager 🚀

A modern productivity tool built to organize tasks intelligently and reduce clutter.

---

## 📌 Overview

AI Task Manager helps users:
- Automatically categorize tasks  
- Suggest priority levels  
- Track productivity trends  

It focuses on simplicity, speed, and clean UI.

---

## 🛠 Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- MongoDB
- OpenAI API

---

## ✨ Features

### 1. Smart Categorization
Tasks are grouped automatically based on content.

### 2. Priority Detection
The system predicts urgency using AI logic.

### 3. Minimal UI
Distraction-free design with smooth animations.
