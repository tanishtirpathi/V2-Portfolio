# Playlistr

#### A modern playlist sharing platform where users can create, discover, and share their favorite Spotify playlists.

---

## Overview

Playlistr is a community-driven platform designed for music lovers who enjoy discovering new sounds and sharing their curated vibes.

It allows users to showcase their Spotify playlists, explore collections created by others, and connect through music preferences.

The platform is currently in development, with 40+ active users already waiting for the official release.

---

## Tech Stack

- React  
- Node.js  
- Express  
- MongoDB  
- Spotify API  
- Tailwind CSS  

---

## Why I Built This

Music has always been personal, but discovering new playlists often feels disconnected.

I wanted to create a space where curated Spotify playlists could be shared more intentionally — not just links, but collections with personality.

This project also gave me the opportunity to work with third-party APIs (Spotify), user-based data handling, and scalable backend structure.

---

## Features

- Create and publish Spotify playlists  
- Discover playlists shared by other users  
- Community-driven music discovery  
- Clean and minimal interface  
- Fast and responsive experience
